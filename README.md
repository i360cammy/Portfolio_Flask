# Cameron's Portfolio Website

A simple Flask-based portfolio website.

## Setup

1. Create virtual environment: `python -m venv .env`
2. Activate it: `.\.env\Scripts\Activate.ps1` (Windows)
3. Install dependencies: `pip install -r requirements.txt`
4. Run the app: `python app.py`
5. Open http://127.0.0.1:5000/

## Features

- Header with navigation (hides on scroll down, shows on scroll up)
- Intro section with animated text
- Projects carousel
- Footer with social links
